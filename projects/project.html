2023
Personal project
Programming language: Backend: Django, Frontend: SwiftUI
Duration: In progress

This App is a Notes App with integrated Chatgpt support. The User can upload PDFs from his device and structure them in different directories. The user can manually draw on the uploaded file and make notes. When something on the notes is not clear, the user can mark it and with the help of Apples Optical Character Recognition (OCR), the marked area is translated into text and sent to the Chatgpt API, where the unclear text section is processed and explained to the user.

Personal project
2023
Programming language: Backend: Django, Frontend: ReactJS
Duration: In progress

In this mobile app named FindUrSoulmate the user can create questions with two respective answer options. At the same time, the user can answer questions that were created by other users. The questions that are shown to the user are based on a personality profile that is created based on the user's answers to the question. In the score table are shown users with the most similar personality profiles. The personality profiles are created via embedding comparisons and LLMs.
